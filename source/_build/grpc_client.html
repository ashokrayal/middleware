grpc_client module¶

grpc_client.run()¶

Run gRPC client to interact with the Product Service.

Connects to the Product Service using gRPC, performs PlaceOrder and UpdateOrder operations, and logs the responses.

Raises:

grpc.RpcError: If an error occurs during gRPC communication.

Returns:

None
